# 🐛 Error Museum

*Every error is a lesson. This is where I document them all.*

---

## How to Document an Error

1. **Screenshot it** (or copy the full message)
2. **Note what you were trying to do**
3. **Document how you fixed it**
4. **Save it here** with a descriptive filename

**File format:** `error-[date]-[short-description].md`

---

## Error Template

```markdown
# Error: [Short Description]

## 📅 Date Encountered

## 🤔 What I Was Trying to Do

## ❌ The Error Message

## ✅ How I Fixed It

## 💡 What I Learned
```
