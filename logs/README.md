# 📓 Learning Logs

*My journey, documented in my own words.*

---

## How to Use This

Write regular entries about your learning journey. Be honest, be real.

**Entry format:** `log-YYYY-MM-DD.md` or `week-01.md`

---

## Entry Template

```markdown
# Learning Log - [Date/Week]

## 📅 Time Period

## 🎯 What I Set Out to Learn

## 📝 What I Actually Learned

## 🤯 Biggest "Aha!" Moment

## 😤 Biggest Struggle

## 🏆 Small Win I'm Proud Of

## 🤔 Questions I Still Have

## 🎯 Next Steps
```
